# Survey Data Files

This directory contains downloadable datasets referenced throughout the JTBD book content.

## Available Datasets

### Chapter 4: Core Functional Job Analysis

- `notion-survey-raw-data.csv` - 300 Notion users rating 22 functional capabilities (5-point Likert scale)
- `factor-analysis-results.csv` - Processed factor analysis results showing 4-factor structure
- `factor-loadings-matrix.csv` - Complete factor loadings matrix with rotation results

### Research Methodology Files

- `maxdiff-survey-data.csv` - MaxDiff analysis survey responses for job prioritization
- `user-interview-demographics.csv` - Interview participant demographics and segmentation data
- `survey-instrument.csv` - Complete survey questions and response scales used

## File Formats

All files are in CSV format for maximum compatibility with:

- R (read.csv())
- Python (pandas.read_csv())
- Excel/Google Sheets
- Statistical software (SPSS, SAS, etc.)

## Data Usage

These datasets support the research examples and statistical analyses presented in the book. Readers can:

- Replicate the analyses shown
- Explore alternative analytical approaches
- Use as practice data for learning JTBD methods

## Ethics & Privacy

All data has been simulated. No personally identifiable information is included.
